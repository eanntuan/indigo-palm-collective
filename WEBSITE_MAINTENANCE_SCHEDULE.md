# Website Maintenance Schedule

**Site:** indigopalm.co
**Owner:** Indigo Palm Collective
**Last Updated:** February 25, 2026

---

## Monthly Tasks (1st of each month)

**Duration:** ~30 minutes

### Content Review
- [ ] Update meta descriptions if property features changed
- [ ] Add new content/blog posts (if applicable)
- [ ] Update seasonal messaging (winter/summer/events)

### Performance Check
- [ ] Review Google Search Console for:
  - New keyword opportunities
  - Search ranking changes
  - Click-through rates
- [ ] Check Google Analytics traffic trends

### Social Media
- [ ] Update Pinterest boards with new photos
- [ ] Verify Pinterest verification still active

**Agent:** website-seo-monitor can help with technical checks

---

## Quarterly Tasks (Jan 1, Apr 1, Jul 1, Oct 1)

**Duration:** ~1-2 hours

### Technical Audit
- [ ] Check for broken links (all pages)
- [ ] Review page load speed (Google PageSpeed Insights)
- [ ] Verify all images have alt text
- [ ] Test mobile responsiveness
- [ ] Check SSL certificate validity

### Content Optimization
- [ ] Update property photos (seasonally appropriate)
- [ ] Refresh testimonials/reviews on site
- [ ] Review and update pricing displays
- [ ] Check competitor websites for ideas

### SEO Health Check
- [ ] Run full SEO audit (Lighthouse or similar)
- [ ] Review backlinks (who's linking to you)
- [ ] Check sitemap.xml is current
- [ ] Verify schema markup is valid

### Analytics Review
- [ ] Review traffic sources (organic, direct, social)
- [ ] Identify top-performing pages
- [ ] Analyze bounce rates by page
- [ ] Set goals for next quarter

**Agent:** website-seo-monitor can run technical audits

---

## Annual Tasks (January)

**Duration:** ~half day

### Major Review
- [ ] Full website redesign assessment
- [ ] Review all copy for accuracy
- [ ] Update all property descriptions
- [ ] Refresh all photography
- [ ] Review competitor positioning

### Strategic Planning
- [ ] Set SEO goals for the year
- [ ] Plan content calendar
- [ ] Identify new keyword targets
- [ ] Review hosting/infrastructure costs
