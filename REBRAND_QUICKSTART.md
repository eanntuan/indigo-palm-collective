## ❓ Questions?

**Q: Can I rebrand sooner?**
A: Only if renovations + photos are done. Brand promise must match reality.

**Q: What if I want to test the rebrand first?**
A: Run on a test branch:
```bash
git checkout -b test-terra-luz
bash rebrand-to-terra-luz.sh
# Review, then delete branch or merge
```

**Q: Can I undo the rebrand?**
A: Yes! Script creates backup branch:
```bash
git checkout backup-casa-moto-[DATE]
```

**Q: Will this affect bookings?**
A: Website changes are instant. Airbnb listing keeps existing reservations under old name, but new bookings see new brand.
