## Exact Canva Template Specs

### **Template Setup:**

1. **Go to Canva** → Create a design → Custom size
2. **Dimensions**: 1000 x 1500 pixels (2:3 ratio - Pinterest optimized)
3. **Template name**: "Indigo Palm Pinterest Pin Template"

### **Layout Structure:**

**TOP SECTION (200px high):**
- Background: Cream/Beige rectangle (#F5F1E8)
- Logo: Upload Indigo Palm icon (or use simple palm leaf icon from Canva)
  - Size: 60px x 60px
  - Position: Centered, 30px from top

**MIDDLE SECTION (1100px high):**
- Your property photo
- No border, full bleed

**TEXT OVERLAY ON PHOTO (bottom 400px):**
- Semi-transparent overlay: Black rectangle at 30% opacity
- Property name in ALL CAPS
  - Font: **Montserrat Bold** or **Poppins Bold**
  - Size: 60pt
  - Color: White (#FFFFFF)
  - Letter spacing: +10
  - Position: Centered, 300px from bottom

- Tagline below property name
  - Font: **Lora Regular** (serif) or **Montserrat Light**
  - Size: 24pt
  - Color: White (#FFFFFF)
  - Letter spacing: +5
  - All lowercase
  - Position: Centered, 220px from bottom

**BOTTOM SECTION (Watermark):**
- Text: "IndigoPalmCollective.com"
  - Font: **Montserrat Regular**
  - Size: 18pt
  - Color: White (#FFFFFF)
  - Background: Dark brown/black bar (#2C2420) full width, 60px high
  - Position: Bottom of image

### **Example Layout:**

## How to Execute in Canva (Step-by-Step)

### **Phase 1: Create Template (20 minutes)**

1. Open Canva → "Create a design" → "Custom size" → 1000 x 1500px
2. Add cream rectangle at top (200px high, #F5F1E8)
3. Add palm leaf icon or upload your logo (centered at top)
4. Save as "Indigo Palm Pinterest Template"

### **Phase 2: Make Your 12 Pins (40 minutes)**

For each pin:
1. Duplicate template
2. Upload your photo (from file paths above)
3. Drag photo into middle section
4. Add dark overlay rectangle at bottom (30% opacity)
5. Add property name text (Montserrat Bold, 60pt, white, all caps)
6. Add tagline text (Lora Regular, 24pt, white, lowercase)
7. Add watermark at bottom: "IndigoPalmCollective.com"
8. Download as PNG
9. Name file: `[property-name]-pinterest-[number].png`
   - Example: `terra-luz-pinterest-1.png`

### **Phase 3: Upload to Pinterest (15 minutes)**

1. Go to Pinterest.com
2. Click "Create" → "Create Pin"
3. Upload image
4. Copy-paste title from above
5. Copy-paste description from above
6. Add destination link: Your website property page
7. Select board: "Indigo Palm Collective Properties" (create if needed)
8. Publish

### **Phase 4: Schedule Future Pins (10 minutes)**

Use Later.com or Planoly:
- Post 2-3 pins per week
- Repin successful pins every 2-3 weeks with different titles
- Track saves + clicks in Pinterest Analytics
