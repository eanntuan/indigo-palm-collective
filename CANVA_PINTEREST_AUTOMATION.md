## 🎯 THE "MAGIC" FEATURES YOU'LL USE

### 1. **Brand Templates** (Your Foundation)
- Pre-designed templates with locked brand elements
- Only text/images change, colors/fonts/logos stay consistent
- Creates visual consistency across all properties

### 2. **Magic Design/Magic Switch**
- Takes ONE design and auto-generates variations
- Changes layouts while keeping your content
- Creates 8-10 different pin styles from one template

### 3. **Bulk Create**
- Upload CSV file with 50+ headlines and property photos
- Canva auto-generates unique pins for each row
- Perfect for A/B testing headlines

### 4. **Content Planner**
- Schedule pins directly to Pinterest from Canva
- No downloading/uploading files
- Schedule 30 days of content in one session

## 🎨 DESIGN VARIATIONS TO TEST

### A/B Test These Layouts

Canva's guide recommends testing these pin styles:

**1. Photo-Dominant (Hero Style)**
- 80% photo, 20% text overlay
- Works for: Stunning property exteriors, pool shots, mountain views
- Best for: Aspirational/emotional appeal

**2. Split Layout (50/50)**
- 50% photo, 50% text/color block
- Works for: When you have a key benefit to highlight
- Best for: "5 Reasons to Stay at Casa Moto" style pins

**3. Text-Heavy (Informational)**
- 30% small photo, 70% text list
- Works for: "Ultimate Palm Springs Packing List," "What to Do Near Cozy Cactus"
- Best for: Keyword-rich searches, educational content

**4. Before/After or Gallery Style**
- Multiple small photos in grid
- Works for: Showing different rooms, day-to-night transformations
- Best for: Demonstrating variety/value

**5. Video Pins** (Advanced)
- 15-second property tour
- Square format (1:1 ratio, 1000x1000px)
- Works for: Showing property flow, highlighting amenities
- Best for: Higher engagement (video pins get 3x more saves)

**Use Magic Design to auto-generate all 5 variations** from your master template.

**What Are Story Pins?**
- Up to 20 pages per pin
- Swipe-through format (like Instagram Stories)
- Great for tutorials, guides, property tours

## 💡 PRO TIPS FROM CANVA'S GUIDE

### 1. **Seasonal Content = 45 Days Ahead**
- Create Coachella pins in early March (not April)
- Summer pool content in late April
- Pinterest users plan WAY ahead

### 2. **Keywords in File Names**
- Before uploading to Canva, rename files:
  - "casa-moto-palm-springs-vacation-rental-pool.jpg"
  - NOT "IMG_1234.jpg"
- Pinterest reads file names for SEO

### 3. **Pin Description Formula**
```
[Emotional Hook] + [Key Features] + [Location] + [Call to Action] + [Hashtags]

Example:
"Unwind in a Cuban-inspired desert sanctuary. Casa Moto blends 1960s mid-century modernism with vibrant Latin warmth. Private pool, indoor/outdoor living, 5 minutes from downtown Palm Springs. Book your escape at Indigo Palm Collective. #PalmSpringsVacationRental #MCMHome #TerraLuz #DesertGetaway"
```

### 4. **Vertical Format ALWAYS Wins**
- 1000x1500px gets 3x more saves than square
- Mobile users scroll vertically
- Takes up more screen real estate = more attention

### 5. **Fresh Pins > Repinning**
- Pinterest rewards NEW content, not repins
- Create 12+ NEW pins per month minimum
- Don't just repin the same image with different text

### 6. **Rich Pins = Better Performance**
- Enable Rich Pins on your website (shows live price, availability)
- Requires website metadata setup
- 50% higher click-through rate vs regular pins

## 🚨 COMMON MISTAKES TO AVOID

### ❌ Don't Do This:

1. **Creating pins manually one by one**
   - Use Bulk Create for efficiency
   - Manual = 30 min per pin, Bulk = 2 min per pin

2. **Ignoring keywords**
   - Pinterest is a SEARCH engine
   - Users search "Palm Springs vacation rental" not "Casa Moto"
   - Use keywords in: Pin title, description, board name, file name

3. **Same image, different text**
   - Pinterest penalizes duplicate content
   - Each pin needs unique photo OR unique layout

4. **Not connecting Content Planner**
   - Scheduling from Canva saves 90% of time
   - No downloading/uploading/manual posting

5. **Forgetting to add website link**
   - Every pin should link to property page
   - This is how you convert saves → website visits → bookings

6. **Using low-quality photos**
   - Pinterest is VISUAL
   - Blurry photos = instant scroll-past
   - Invest in pro photography (you already have this!)

7. **Generic headlines**
   - "Beautiful Home in Palm Springs" = BORING
   - "Where Cuban Soul Meets Desert Modernism" = COMPELLING
